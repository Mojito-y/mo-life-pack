# Mo Life Pack

Clone-and-run pack for **Mo Coach** and future AI-in-daily-life tools.

## What lives here

- `skills/mo-coach`: installable Codex skill
- `packages/setup-cli`: one-command setup flow
- `packages/mo-coach-core`: plan generation and coach config logic
- `apps/lark-bot`: Feishu/Lark bot starter
- `templates/`: coach and environment defaults

## Quick start

```bash
pnpm install
pnpm setup
```

## Goal

Package a personalized fitness coach that can be renamed, styled, and adapted to:

- fat loss
- muscle gain
- strength
- conditioning
- mobility
- recovery
- general fitness

The repo is structured so other daily-life agents and tools can be added later without reshaping the base.
