# 输出模板

## 飞书短答

```markdown
结论：一句话判断；证据门禁未完成时可以明确写“当前无法判断”。

截至：YYYY-MM-DD HH:mm，时区；数据是否延迟。

已证实：
- 最重要的 2–3 条事实。

关键反证：
- 最可能推翻判断的一条证据。

行动门槛：
- 满足什么条件才观察、建仓、加仓、减仓或退出。个性化门禁未完成时写“待确认 / 暂不操作”，不输出仓位数字。

风险与未知：
- 尚未验证且会改变结论的事项。

下一步：
- 一个具体动作和复核日期。
```

## 完整个股研究

```markdown
# [公司 / 代码] 研究

## 结论
## 数据截止与来源
## 投资逻辑
## 基本面与财务质量
## 熊 / 基准 / 牛估值情景
## 最强反方观点与亏损路径
## 催化剂、风险与未知
## 决策卡
## 监控指标与复核日期
## 来源
```

## ETF 比较

```markdown
## 一句话结论
## 两只产品的目标暴露是否相同
## 指数、持仓与集中度
## 费率、跟踪、流动性与规模
## 币种、汇率、分红与税务待核对项
## 适用情景与不适用情景
## 配置区间和再平衡条件（个性化门禁未完成时写待确认 / 暂不操作）
## 来源与数据日期
```

## 组合复盘

```markdown
## 一句话结论
## 组合快照
## 收益归因
## 集中度与风险暴露
## 逻辑成立 / 待验证 / 已破坏
## ETF 重叠与币种风险
## 调整优先级与无操作方案
## 再平衡规则
## 下次复盘日期与触发条件
```
