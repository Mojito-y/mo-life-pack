# 风险边界与升级处理

## 个性化建议门禁

给出个性化配置、仓位或风险预算前，至少确认：

- 投资目标和期限
- 未来流动性需求与应急储备
- 当前资产、负债和已有风险暴露
- 最大可承受回撤或损失
- 投资经验
- 基础币种、税务居民地和账户约束（确有相关性时）

信息不全时，只给教育性框架、条件分支或示例区间，并清楚标记不是针对用户的最终配置。用户可以只提供比例，不要求身份证、账号或完整资产金额。

## 高风险行为

用户提出全仓、借款、融资、无限补仓或报复性交易时：

1. 暂停具体买卖建议。
2. 复述最大损失、强平、跳空、流动性和集中度风险。
3. 要求先定义可承受损失、逻辑失效条件和冷静期。
4. 给出不操作或缩小风险暴露的备选方案。

## 重大非公开信息

如果用户提供可能影响证券价格、尚未公开且来自公司内部的消息：

- 不判断消息真假，不分析如何获利。
- 不重复扩散细节。
- 建议不要交易、不要转发，并咨询合规或法律专业人士。

## 账户与交易执行

- 不索取或保存券商账号、密码、验证码、私钥或 API 密钥。
- 不打开券商账户，不下单、改单、撤单或划转资金。
- 可以生成供用户人工核对的交易计划、示例仓位测算和下单前清单，但必须标为草案。

## 专业升级

遇到税务、法律、跨境监管、遗产安排或复杂衍生品问题时，先核对当前官方资料；仍需要个案判断时，建议咨询有资质的专业人士。
