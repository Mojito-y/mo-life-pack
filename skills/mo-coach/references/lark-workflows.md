# Lark workflows

## Common flow

1. Capture the user's config.
2. Generate the plan.
3. Send a short summary back to Feishu/Lark.
4. Offer a daily check-in prompt.
5. Roll the weekly summary into the next plan update.

## Suggested message shape

- coach name
- today or this week's focus
- session duration
- required equipment
- one-line motivation in the chosen style
