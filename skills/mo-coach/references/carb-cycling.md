# 碳循环知识储备

## 目标

碳循环用于在减脂阶段兼顾热量缺口、训练表现、恢复和长期执行。Mo Coach 应把它当作饮食安排策略，而不是医学治疗或极端节食方案。

## 基本原则

- 高碳日绑定大肌群、高强度或关键训练日，优先支持训练表现和糖原恢复。
- 中碳日用于普通训练日或活动量中等的工作日，兼顾执行和恢复。
- 低碳日绑定休息日、轻有氧或恢复日，用于制造温和热量缺口。
- 蛋白质保持稳定，常用范围为 1.6-2.2 g/kg/天；减脂期优先保护肌肉和饱腹感。
- 脂肪与碳水反向调整：高碳日脂肪相对收低，低碳日脂肪适度上调，但不极端。
- 饮水和电解质要跟上，训练日、高碳日和出汗多时适当增加饮水。

## 输出规则

当用户要求减脂计划、每日吃什么、训练日饮食或体重平台期调整时：

1. 先确认目标、体重、训练日、训练强度、饮食限制和最近体重趋势。
2. 按训练安排标记高碳/中碳/低碳日。
3. 每天给出蛋白质优先级、主食时机、蔬菜和饮水建议。
4. 如果没有体重或热量数据，用“餐盘法”和相对份量表达，不假装精确。
5. 如果出现头晕、异常疲劳、暴食冲动、睡眠明显变差或训练表现连续下降，优先降低激进程度。

## 飞书短回复形态

- 今日类型：高碳日 / 中碳日 / 低碳日
- 今日重点：训练表现 / 稳定执行 / 恢复和缺口
- 吃法：蛋白质、主食、蔬菜、脂肪、水
- 训练：主训练或恢复安排
- 调整依据：体重趋势、饥饿感、疲劳和完成度
